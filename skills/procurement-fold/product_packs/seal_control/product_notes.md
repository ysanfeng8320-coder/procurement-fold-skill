# 智能印章 / 印控仪 — 产品说明摘要

## 产品线（软硬一体）

| 终端 | 角色 |
|------|------|
| 用印宝 / 智能章筒 | 实体公章加智能锁；外带、授权用印、违规锁章、轨迹 |
| 印控台 | 管控多枚章筒；双摄、OCR/文件校验、登录验证 |
| 印章柜 / 用印柜 | 集中存放、盘点；可与传统章混放（RFID） |
| 智能盖章机（如 G5200） | 固定场景自动盖章、人印隔离、多章共管、水印/OCR |
| 天鉴等印控平台 | 审批、归档、风控、报表、对接 OA/政务 |

## 核心价值

- 用印全流程可追溯（人、时、地、文件、影像）
- 人印隔离与违规干预（电子围栏、锁章、防换页等）
- 物理章与电子章统一监管（视方案）

## 典型场景 / 客群（销售语境）

- 基层治理 / 村居公章（纪委牵头常见）
- 公安、机关集中管章
- 政务服务、组织/社工、信访
- 大型企业集中用印（美的、阿里、万达、贝壳等——企业线，政采折叠时作对照）
- 高校、部队等（视项目）

## 与「电子印章」关系

电子印章/电子签章常为关联品或软件包；硬件印控（盖章机/用印宝/印控台）可能尚未采购 → 典型 **assoc_gap** 候选。

## 检索别名（公告用词）

印控仪、印控台、智能印章、智慧印章、盖章机、用印宝、用印柜、印章柜、印章管理系统、用印管控、电子印章（关联）。
